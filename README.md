# 🚗 Car Rental System — Java Desktop Application

A fully functional Car Rental Management System built with Java Swing for the OOP course at SZABIST University.

## Features
- Staff login & authentication
- Dashboard with live statistics
- Vehicle management (Car, SUV, Van)
- Customer registration with validation
- Rental booking with auto invoice
- File-based data persistence

## Tech Stack
Java JDK 17 | Java Swing | Object Serialization | IntelliJ IDEA

## How to Run
1. Clone the repo
2. Open in IntelliJ IDEA
3. Run `MainApp.java`
